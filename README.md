Projeto de Análise Comercial
Este projeto consiste em uma análise exploratória de dados de uma operação comercial, utilizando a linguagem Python e a biblioteca Pandas. O objetivo central é extrair indicadores de desempenho que fundamentem a tomada de decisão estratégica através da consolidação de múltiplas bases de dados.

Escopo do Projeto
A análise abrange o processamento de dados de faturamento, base de clientes, folha salarial e logística de entregas. A solução automatiza o cálculo de métricas essenciais para a gestão do negócio, eliminando processos manuais e reduzindo a margem de erro.

Funcionalidades Implementadas
Consolidação de dados de diferentes fontes para uma visão unificada da operação.

Cálculo de faturamento total e performance por unidade de negócio.

Análise de produtividade da força de vendas (Ticket Médio e Volume de Vendas).

Mensuração de eficiência logística e análise de devoluções.

Identificação de percentual de funcionários por setor para otimização de recursos.

Tecnologias Utilizadas
Linguagem Python.

Biblioteca Pandas para manipulação e tratamento de dados.

Metodologia Aplicada
Carregamento e inspeção inicial das bases de dados.

Limpeza e normalização de registros para garantir a integridade da análise.

Cruzamento de informações (merge) entre tabelas de vendas e funcionários.

Agrupamento e sumarização de dados por categorias específicas.

Exportação dos resultados processados para análise executiva.
